# 第13章 库

> And if I really wanted to learn something I'd listen to more records. / And I do, we do, you do.
> —The Hives, "Untutored Youth"

本章介绍几个让生活更轻松的库。C 库这些年变得不那么学究了。十年前典型库只提供工作所需的最小工具集，期望你从这些基础构建方便、对程序员友好的版本。本章介绍的库都提供「简单」接口，如 cURL 的 `curl_easy_*` 函数或 SQLite 执行数据库事务所有步骤的单个函数。

## GLib

标准库留下许多空白，自然会有库来填补。GLib 实现了足够多的基本计算需求，移植到几乎所有平台（包括无 POSIX 的 Windows），且足够稳定可依赖。

本书中已多次使用 GLib：链表（示例 2-2）、测试框架、Unicode 工具、哈希表、正则表达式等。还有事件循环、文件工具、配置文件解析器等。

## POSIX

POSIX 标准向标准 C 库添加若干有用函数。值得了解的两个突出部分：正则表达式解析和将文件映射到内存。

### 解析正则表达式

正则表达式是表达文本模式的手段。POSIX 定义两种：基本（BRE）和扩展（ERE）。PCRE（Perl 兼容正则）通过 libpcre 提供，GLib 有高层封装。

四步流程：1) regcomp/pcre_compile 编译；2) regexec/pcre_exec 执行；3) 用返回的偏移提取子串；4) regfree 释放。

### mmap

将文件映射到内存，避免一次性读入。POSIX 的 `mmap` 在 Windows 上对应 CreateFileMapping 和 MapViewOfFile。

## SQLite

轻量级嵌入式数据库，单文件，无需服务器。适合配置、缓存、本地数据存储。

## GNU Scientific Library (GSL)

数值计算：线性代数、统计、特殊函数、数值积分等。

## libxml2

XML 解析与生成。DOM 和 SAX 风格 API。

## libcURL

URL 传输库，支持 HTTP、HTTPS、FTP 等众多协议。`curl_easy_*` 接口简单易用。
