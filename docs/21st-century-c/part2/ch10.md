# 第10章 更好的结构体

> Twenty-nine different attributes and only seven that you like.
> —The Strokes, "You Only Live Once"

本章讲接受结构化输入的函数，以及改善库的用户界面。涵盖 C99 引入的三种语法：复合字面量（compound literals）、可变长宏和指定初始化器（designated initializers）。

## 复合字面量

可向函数发送字面值：`f(a_value)`。但要发送元素列表——如 `{20.38, a_value, 9.8}` 的复合字面量——需在复合字面量前加类型转换，否则解析器会困惑：

```c
f((double[]) {20.38, a_value, 9.8});
```

复合字面量自动分配，无需 malloc 或 free。在出现作用域结束时自动消失。

**示例 10-1. 用复合字面量绕过临时变量（sum_to_nan.c）**

```c
#include <math.h>
#include <stdio.h>
double sum(double in[]){
    double out=0;
    for (int i=0; !isnan(in[i]); i++) out += in[i];
    return out;
}
int main(){
    double list[] = {1.1, 2.2, 3.3, NAN};
    printf("sum: %g\n", sum(list));
    printf("sum: %g\n", sum((double[]){1.1, 2.2, 3.3, NAN}));
}
```

## 可变长宏

`__VA_ARGS__` 展开为给定的任意元素集。结合复合字面量可写出接受任意长度列表的函数接口。

## 指定初始化器

不用 `person_struct p = {"Joe", 22, 75, 20}` 这种难读易错的形式，可写：

```c
person_struct p = {.name="Joe", .age=22, .weight_kg=75, .education_years=20};
```

## 从函数返回多个值

现在初始化结构体不痛苦了，从函数返回结构体也无痛，能大大澄清函数接口。返回结构体可包含多个输出和错误状态。

## 可选和命名参数

通过将一切包在可变长宏中，可写接受可变数量命名参数的函数，并为用户未指定的参数赋默认值。例如贷款计算器：`amortization(.amount=200000, .rate=4.5, .years=30)` 和 `amortization(.rate=4.5, .amount=200000)` 都有效，后者使用默认 30 年期限。

## foreach

用可变长宏生成数组（以哨兵结尾）的复合字面量，然后 for 循环对每个元素应用函数，可实现 foreach 风格接口。
